# Example: Python Demo

这是一个最小示例目录，用来说明这个模板如何和真实 Python 项目一起放置。

## 建议流程

1. 在这个目录下初始化你自己的 Python 项目
2. 把仓库根目录的 `docs/`、`templates/`、`.ai/`、`tools/` 作为工作流骨架
3. 使用 `templates/01-proposal.prompt.md` 到 `templates/05-coding.prompt.md` 推进

## 这里只提供结构示意，不强行内置一整套依赖

你可以按自己习惯使用：

- `uv`
- `venv + pip`
- `poetry`
- `pdm`

关键不是包管理器，而是文档与任务流保持稳定。
